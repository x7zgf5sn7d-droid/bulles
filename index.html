📋 Format attendu du tableur

Ligne 1 = en-têtes, données à partir de la ligne 2 :

ColonneValeurs
A — PrénomTexte libre
B — NomTexte libre
C — Rôleencadrant / plongeur / eleve
D — NiveauE1–E4, MF1, MF2, PE20, N1–N4…
E — LicenceNuméro
F — Certificatvalide / expire
G — Activitétechnique / exploration
📖 Guide complet — Google Sheets sans clé API
✓ Principe : tu publies ton Google Sheet en CSV public. ApsaVO Bulles lit ce fichier directement — aucun compte Google Cloud, aucune clé API, totalement gratuit.
1
Créez un nouveau Google Sheet
Allez sur sheets.google.com et créez un tableur vide.
En bas, renommez l'onglet (double-clic sur "Feuille 1") en Plongeurs.
Ligne 1 — en-têtes obligatoires (une par colonne) :
A1: Prenom  |  B1: Nom  |  C1: Role  |  D1: Niveau  |  E1: Licence  |  F1: Certificat  |  G1: Activite
2
Remplissez vos plongeurs à partir de la ligne 2
Exemples de valeurs :
Col. C (Role) : encadrant / plongeur / eleve
Col. D (Niveau) : PE20 PE40 PA40 N1 N2 N3 E1 MF1…
Col. F (Certificat) : valide ou expire
Col. G (Activite) : exploration ou technique
3
Publiez le tableur en CSV
Dans le menu Google Sheets :
① Cliquez sur Fichier (en haut à gauche)
② Sélectionnez Partager → Publier sur le web
③ Dans la fenêtre : choisissez l'onglet Plongeurs (pas "Document entier")
④ Dans le menu déroulant du format : choisissez Valeurs séparées par des virgules (.csv)
⑤ Cliquez Publier puis confirmez avec OK
4
Copiez l'URL CSV générée
Après publication, Google affiche une URL longue. Elle ressemble à :
https://docs.google.com/spreadsheets/d/VOTRE_ID/pub?gid=0&single=true&output=csv ⚠️ Copiez l'URL complète — elle doit contenir output=csv ou format=csv.
5
Collez l'URL dans ApsaVO Bulles et synchronisez
① Allez sur l'onglet 📊 Synchronisation
② Collez l'URL dans le champ "URL CSV du Google Sheet"
③ Cliquez 💾 Sauvegarder l'URL (mémorisée pour les prochaines fois)
④ Cliquez 🔄 Synchroniser — les plongeurs s'importent automatiquement
⑤ Les doublons sont détectés et ignorés
🔄 Mettre à jour la liste
Modifiez le Sheet → revenez dans l'app → cliquez Synchroniser.
Seuls les nouveaux plongeurs sont ajoutés.
📱 Partager l'application
Envoyez le fichier ApsaVO_Bulles.html par email ou WhatsApp.
Sur mobile : ouvrir → Partager → Écran d'accueil.
